# Sistema de Gerenciamento de Coleção Biológica — UFJ

Trabalho final da disciplina de **Programação Orientada a Objetos**, desenvolvido para informatizar o processo de curadoria de exemplares do Laboratório de Biodiversidade da Universidade Federal de Jataí (UFJ).


## Sobre o projeto:

O Laboratório de Biodiversidade da UFJ mantém uma coleção científica de espécimes biológicos provenientes de pesquisas, inventários ecológicos, monitoramentos ambientais e projetos de conservação. Cada exemplar recebe um número de tombo único, e precisa ter seus dados taxonômicos, geográficos e de coleta devidamente registrados.

Este projeto implementa um sistema em **Java (Swing)**, seguindo o padrão **DAO**, que permite gerenciar essa coleção de forma segura, organizada e extensível.
